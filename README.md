# Employee_Managament_System
Java Spring Boot CRUD Employee Management System

## 📌 Overview
The Employee Management System is a backend application built using **Java, Spring Framework, Spring Boot, and MySQL**.  
It provides a simple and efficient way to manage employee records with full CRUD functionality.

## ⚙️ Features
- Add, update, view, and delete employee records
- Integrated with MySQL database for persistent storage
- Layered architecture for modularity and scalability
- Utilizes Spring annotations for dependency injection and configuration
- Developed using IntelliJ IDEA for efficient coding and debugging

## 🛠 Tech Stack
- **Programming Language:** Java  
- **Frameworks:** Spring Framework, Spring Boot  
- **Database:** MySQL  
- **IDE:** IntelliJ IDEA
